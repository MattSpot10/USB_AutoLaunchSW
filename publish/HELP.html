Description

This software serves as a USB event-triggered application launcher and terminator. It automatically launches specified applications when a USB device is inserted and optionally terminates them when the device is removed.

Configuration

To configure the software, click the Configure button in the system tray menu. This will open a JSON configuration file.

Obtaining USB Information

To obtain the USB Vendor ID (VID), Product ID (PID), and Serial Number (if applicable), follow these steps:

  1. Open Task Manager.

  2. Select your desired USB device.

  3. Right click and select Properties, then click the Details tab.

  4. In the Property dropdown, choose Hardware Ids to view the device information.

Configuration File Notes

  • If a property in the configuration file is left empty (i.e., ""), that property will not limit the trigger event, meaning the software will match all devices based on other specified properties (e.g., Vendor ID or Product ID).

  • You can define multiple USB devices and desired executables by adding more entries within the USBSettings array. Enclose each set of properties within curly braces {} and separate them with commas.

  • The RunningProcess field in each configuration entry should not be edited. It is used to manage processes during execution and should be included for all instances, but left empty for this configuration.

Example Configuration

{
  "USBSettings": [
    {
      "VendorID": "VID_0483",
      "ProductID": "PID_5740",
      "SerialNumber": "",
      "ExecutablePath": "C:\\Program Files (x86)\\EMBO\\EMBO.exe",
      "StartOnInsert": true,
      "KillOnRemove": true,
      "RunningProcess": ""
    },
    {
      "VendorID": "VID_1234",
